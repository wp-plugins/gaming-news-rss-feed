== Description ==

Adds a customizable widget which displays the latest Game related news from all over the world. It can be integrated anywhere in the blog. This newsticker shows up the last five or more Game related news posts from sone of the worlds most popular Gaming sites. This is a very nice solution for all those wanting to share breaking gaming related news and for all the gaming blogs out there.

If you have tech related questions/support etc, though I don't see why you would (check the read me file included for help), it's really a basic plugin, shoot me a msg on my GuyRo blog @ guyro.com in the comments and i'll do my best to get back to you.

*Feature List*

* customizable widget
* displays a user-defined number of links
* title shortening

== Installation ==

*Could not be simpler*
1. Extract `games-rss-plugin.zip`.
2. Upload the `games-rss-plugin`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.
